## 📂 Highlighted Repositories

I organize my work by technical domains to demonstrate my versatility across the AI and Data Science spectrum.

### 👀 Computer Vision & Generative AI
*   **[Generative Diffusion Models for Point Clouds](https://github.com/YgggC/Image-analysis/blob/main/Project_Diffusion_models/notebooks/JointPointCloudDiffusion.ipynb):** Implemented state-of-the-art joint point-cloud diffusion models from scratch using PyTorch to model complex 2D geometric shape distributions.
*   **[Geospatial Semantic Segmentation](https://github.com/YgggC/AppliedML/blob/main/TensorFlow_Keras_Potsdam_2D_Semantic_Segmentation/TensorFlow_Keras_Potsdam_2D_Semantic_Segmentation.ipynb):** Trained a fully convolutional network (FCN) with ResNet backbones for urban scene understanding on the high-resolution Potsdam dataset, optimizing inference for large-scale geospatial data.
*   **[3D Reconstruction Engine & Image Registration](https://github.com/YgggC/Image-analysis/blob/main/Lab3_3D%20reconstruction/3D%20reconstruction.ipynb):** Developed a robust multi-view 3D triangulation pipeline using non-linear least squares optimization and built feature-based (SIFT) image stitching solutions using classical OpenCV techniques.

### 💬 NLP & Applied LLM Systems
*   **[LLM Retrieval-Augmented Generation (RAG) System](https://github.com/YgggC/AppliedML/blob/main/LangChain_Ollama_RAG_Assistant/LangChain_Ollama_RAG_Assistant_Template.ipynb):** Engineered a document QA pipeline utilizing LangChain, ChromaDB, and local models (Ollama) to perform accurate semantic searches and retrieve context-aware answers over complex legal texts (e.g., the EU AI Act).
*   **[NLP Fundamentals](https://github.com/YgggC/NLP_portfolio):** Implemented baseline text classification pipelines, and explored feature representations and model performance trade-offs  

### 📐 Mathematical ML & Optimization
*   **[Compressed Sensing & Sparse Recovery]((https://github.com/YgggC/AppliedML/blob/main/SciPy_L1Minimization_Compressed_Sensing/SciPy_L1Minimization_Compressed_Sensing.ipynb)):** Developed $L_1$ minimization algorithms using Coordinate Descent for image reconstruction from severely undersampled signals, demonstrating deep knowledge of sparsity and linear algebra.
*   **[Optimization Algorithms for Machine Learning](https://github.com/YgggC/AppliedML/blob/main/NumPy_SGD_SAG_Logistic_Optimization/NumPy_SGD_SAG_Logistic_Optimization.ipynb):** Implemented and systematically benchmarked advanced optimization techniques like Stochastic Average Gradient (SAG) against standard SGD for large-scale logistic regression tasks.
